## 🏗️ Directory Structure

Each problem solution follows this naming convention:
- **NN_ProblemName/** - Problem folder (where NN is the problem number)
  - `main.cpp` - Solution implementation
  - `README.md` - Problem description and approach (optional)

Example structure:
```
HackerRank/
├── 01_ProblemName/
│   ├── main.cpp
│   └── README.md
├── 02_AnotherProblem/
│   ├── main.cpp
│   └── README.md
└── README.md
```

## 🚀 How to Add a Solution

1. Create a new directory with naming format: `NN_ProblemName/`
2. Add your `main.cpp` solution file
3. Optionally add a `README.md` with:
   - Problem description
   - Approach/algorithm explanation
   - Time and space complexity analysis
   - Examples

4. Compile and test:
```bash
cd NN_ProblemName/
g++ -o main main.cpp
./main
```
